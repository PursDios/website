# Ryan Cooper — Personal Portfolio

Personal portfolio and project showcase, hosted on GitHub Pages.

**Live site:** https://pursdios.github.io/website/

## Overview

A static site built with HTML, CSS, and vanilla JavaScript. No frameworks, no build step — just files served directly via GitHub Pages.

## Pages

- **Home** — intro and selected projects
- **Projects** — full project list with status and stack details
- **Hobbies** — the stuff that happens outside of work
- **About** — background, timeline, and how I work
- **Contact** — ways to get in touch

## Stack

- HTML / CSS / Vanilla JS
- [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) · [Inter](https://fonts.google.com/specimen/Inter) · [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) via Google Fonts
- Deployed via GitHub Pages
